I Only Listen To Vinyl

Type Personal Project
Year 2025
Platform Web App
Stack React · Express.js
Overview

Built for people who are tired of algorithms deciding what they listen to. I Only Listen To Vinyl is a curated portal of vinyl-only DJ sets - underground genres, human selection, no recommendations. The app pairs a retro Technics 1200 turntable-inspired UI with YouTube-powered playback, giving the feeling of digging through a record crate on the web. Playlists are hand-curated by BRONSON.

Live App
Features
  • Vintage Technics 1200 turntable-inspired UI - tactile, not digital
  • YouTube-powered playlist player with authentic vinyl aesthetic
  • Six curated genre playlists: House, Techno, Disco, Dub Techno, Deep House & Techno, Hardgroove
  • Draggable, resizable windows for player and info panels
  • Vinyl-inspired visual effects and UI elements throughout
  • Media controls styled after professional audio equipment

Process
  • Noticed vinyl-only DJ sets on YouTube had a loyal following but no dedicated home
  • Designed around the physicality of vinyl - the UI should feel retro and tactile, not like a streaming app
  • Built the full stack app with Claude Code from spec to deployment
