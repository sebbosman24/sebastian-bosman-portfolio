The Process

I started with a spec. Before writing a single line of code I produced a PRD defining the site architecture, design system, interaction model, and content strategy. I then wrote a CLAUDE.md file — a persistent context document that briefed my AI agent on the project, the design rules, and the build order.

From there I worked iteratively, one feature at a time. Terminal boot sequence. Welcome modal. Portfolio page. Seven project pages. Responsive layout. Deployment. Every meaningful change committed with intention. The git history tells the story of the build as clearly as this page does.

I'm not a software engineer. But I can ship.
